搭建博客

Hexo+Github搭建

教程：

小白教程

Hexo

archer

步骤：
  1. 下载node.js
  2. 下载git
  3. 建立仓库，github page
  4. 新建文件夹
  5. 安装hexo
  6. npm install hexo-cli -g
  7. npm install hexo –save
  8. hexo -v
  9. hexo init
  10. npm install
  11. hexo g #生成
  12. hexo d #部署
  13. hexo new “article” #新建文章
导入模板：
  1. Hexo网站上寻找模板下载
  2. git clone到博客文件夹
  3. 在theme文件夹中修改配置
网页title左侧加icon：
  1. jpg转icon
  2. 将icon文件命名为favicon.ico，并放在新建文件夹icon中
  3. 修改themes中的配置文件，重新定向到icon目录 （public是后面生成的，hexo clean之后会被清除）
  4. 刷新网页
新建文章：
  1. hexo new “article”
  2. hexo clean #清楚缓存
  3. hexo g #生成
  4. hexo s #本地预览
  5. hexo d #部署
建立目录标签：
  1. hexo new page “categories” #博客主目录的source文件夹中会出现目录/标签文件夹
  2. 在生成的md文件中加入type: “categories”
  3. hexo new page “tags”
  4. 在生成的md文件中加入type: “tags”
  5. — 设置Front-matter，只能刚好三个，多一个就成了分界线
效果图片：

Bug：
  1. powershell中无法出现git，此时git版本为2.19，重新安装git，选择能够在cmd命令行使用
  2. hexo init 时候git clone的下载很慢：慢慢等；适时按一下回车看看是否有进度；换一个容易登外网的网
  3. hexo d之后读取不到仓库，要先把ssh配置好
  4. 博客生成需要时间，重启之后就发现可以了。
  5. hexo new “中文” 无法创建文章， “eng”才可以。
  6. tool目录无法打开，本地打开没有问题，网页404，没找到办法解决，最后删除仓库重新装
  7. git bash要运行在博客文件夹里
