Hexo+Github搭建

教程：

[小白教程 ] ( https://zhuanlan.zhihu.com/p/33801391 )

Hexo

archer

步骤：
  1. 下载node.js
  2. 下载git
  3. 建立仓库，githubpage
  4. 新建文件夹
  5. 安装hexo
  6. npm install hexo-cli -g
  7. npm install hexo –save
  8. hexo -v
  9. hexo init
  10. npm install
  11. hexo g #生成
  12. hexo d #部署

导入模板：

  1. Hexo网站上寻找模板下载
  2. git clone到博客文件夹
  3. 在theme文件夹中修改配置
Bug：
  1. powershell中无法出现git，此时git版本为2.19，重新安装git，选择能够在cmd命令行使用
  2. hexo init 时候git clone的下载很慢：慢慢等；适时按一下回车看看是否有进度；换一个容易登外网的网
  3. hexo d之后读取不到仓库，要先把ssh配置好
  4. 博客生成需要时间，重启之后就发现可以了。
  5. hexo new “中文” 无法创建文章， “eng”才可以。

1、请确保node版本大于6.2
2、在博客根目录（注意不是archer根目录）执行以下命令：
npm i hexo-generator-json-content --save

jsonContent:
  meta: false
  pages: false
  posts:
    title: true
    date: true
    path: true
    text: false
    raw: false
    content: false
    slug: false
    updated: false
    comments: false
    link: false
