Linux目录结构：

Windows：以存储介质为主，用C盘D盘这些来实现文件管理，然后才是目录，目录没那么重要

Linux：以目录为主，树形目录结构，只有一个根目录

FHS（文件系统层次结构）标准：

第一层规范：

  • 各个目录应该放什么文件
  • /etc 放置设置文件
  • /bin 与 /sbin 放置可执行文件

第二层规范：

  • 针对/user和/var的子目录
  • /var/log 放置系统登录文件
  • /user/share 放置共享文件

Linux循环登录问题：
是在安装NVIDIA的GPU之后出现的，一般来说是显卡的问题，权限的问题
http://www.jianshu.com/p/d45434f28ca0
GPU是显卡的核心
假如你还没有添加显卡驱动的软件源的话，使用如下方式进行安装：
sudo add-apt-repository ppa:graphics-drivers/ppa
sudo apt-get update
下面开始正文
sudo apt-get remove –purge nvidia-*
sudo apt-get autoremove #特别重要
sudo apt-get install -f #特别重要
sudo reboot

通过这个解决了，能登录上了。
